### Prerequisites
1. Have node and npm installed
2. Typescript compiler
3. esbuild (optional)
4. Webpack

### Give it a try!
1. git clone https://github.com/classAndrew/SoftSim.git
2. cd SoftSim
3. npm run esbuild || npm run build && npm start
4. Visit localhost:8080 in a browser
